# grepit

The secure way to to never remember terminal commands.

Wanna know how to use it?

```bash
curl -s https://raw.githubusercontent.com/emergentmethods/grepit/main/install_grepit.sh | bash
```

Use it with

```bash
grepit <search term>
```
